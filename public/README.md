# Images

This directory contains image assets used by **Cool Snippets**.

## Copyright

Some images included in this directory may be protected by copyright or distributed under third-party licenses.

These assets are provided **only** for use within the Cool Snippets project. They **must not** be redistributed, resold, or reused in other projects unless permitted by their original license.

If you intend to use any image outside of this repository, you are responsible for verifying and complying with its respective license.

## Sources

Where possible, the original source of an image should be documented alongside the asset or in the commit history.

## Adding New Images

Before adding a new image, please ensure that it is one of the following:

- Public domain
- Licensed for commercial and open-source use
- Your own original work
- Used with explicit permission from the copyright holder

Avoid committing copyrighted assets unless their license explicitly allows redistribution.

## Removal Requests

If you are the copyright owner of an image included in this repository and believe it has been used in error, please open an issue or contact the repository owner. The asset will be reviewed and, if appropriate, removed promptly.
